# HTML Resume Project

This project is a personal resume webpage built using pure HTML.  
It showcases personal details, skills, experience, projects, education, and contact information in a structured and semantic way.

## Demo
- Open `index.html` in any modern web browser  
- Screenshots of the working resume page are included below

## Setup
1. Open the project folder
2. Double-click `index.html` or open it in a browser

## Usage
- View the About Me section for a brief introduction
- Check Skills to see technical expertise
- Experience and Education are displayed using tables
- Projects section highlights completed work
- Contact details are available at the bottom
